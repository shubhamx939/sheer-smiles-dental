Welcome to the source code for **Sheer Smiles Pediatric Dentistry**, a modern, responsive web application presenting the services, staff, and story of Sheer Smiles—a pediatric dental clinic.

## 🚀 Features

- ✨ Beautiful, modern design with smooth animations
- 🦷 Overview of pediatric dental services offered
- 👦👧 Meet the team and introduction section
- 📱 Fully responsive: works on desktop, tablet, and mobile devices
- ⚡ Interactive UI elements (custom buttons, tilt cards, animated counters)
- 🌙 Clean and accessible layout

## 🛠️ Getting Started

To run the app locally:

### 1. Clone the repository

```bash
git clone https://github.com/shubhamx939/sheer-smiles-dental.git
cd sheer-smiles-dental
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start the development server

```bash
npm start
```

This will open [http://localhost:3000](http://localhost:3000) in your default browser.

---

## 🏗️ Build for Production

To build an optimized static site:

```bash
npm run build
```

To deploy to GitHub Pages (already set up):

```bash
npm run deploy
```
